
**Real-Time Collaborative Drawing App Only for PC 🎨**

A modern web-based drawing application that allows multiple users to draw together on a shared canvas in real time. Built with a focus on simplicity, performance, and seamless collaboration, users can connect from different locations and instantly see each other's drawings without delay.

**Features:**

* Real-time multi-user drawing
* Smooth canvas interaction (mouse & touch support)
* Color picker and adjustable brush size
* Instant canvas sync across devices
* Clean and responsive UI

This project demonstrates real-time communication using WebSockets and an interactive HTML5 Canvas experience.
